### FREECHARGE TEST

Following are the packages used 

1. Express - The holy grail. The genesis.
2. Sequelize - ORM to cut down the querying.
3. bcrypt - To hash the passwords.
4. jwt - The requirement. 


This application contains 3 APIs

1. Register
2. Login
3. Validate token

Install packages

```
npm install
```

To run 

```
node main.js
```
